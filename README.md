# Timetable Website
This is my timetable website. My project is capable of creating tasks and generating tasks to help manage schedules, specially for students. 
The main motivation was that I noticed many of my peers, including myself struggle with managing our schedule, so I decided to create a timetable website to help organzie the workload.

The website allows users to create tasks, organize them into a weekly timetable, and automatically generate schedules. Users can also sync their timetable with Google Calendar for better accessibility.

## Link
You can acess the website here: https://scheduler-mnsx.onrender.com/

## How To Test the Project
1. Open the website using the link above
2. Login using either a regular account or email so you can sync to Google Calendar
3. Click "Add Tasks" to create tasks
4. Select different task types based off their function
5. Click the Generate button to automaically place tasks into the timetable
6. Additionally, click the Sync button to connect to Google Calender 
> Note: "School" task type will generate the typical 8 hour school blocks Monday - Friday

## Teck Stack
**Python** and **Flask** were chosen because they make backend development easy to understand. Flask’s structure allowed the project to be built easily with basic experience. **JavaScript** was used to improve the user experience by making the website more interactive.
