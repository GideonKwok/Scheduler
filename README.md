This is my timetable website. My project is capable of creating tasks and generating tasks to help manage schedule. 
The main motivation was that I noticed many of my peers, including myself struggle with managing our schedule, so I decided to create a timetable website to help.
The website is simple to access: https://scheduler-mnsx.onrender.com/login, this is the link.
It is a weekly timetable.
To use my website, login either with email or just an account, regular account means you can't sync to google calender. 
You can click the "Add tasks" button to add tasks, choose different types of task types for different functions. 
Once added, it could be generated by cliking the button. Once generated, it should be outputed onto the screen and in the timetable.
Additionally, you can sync it to your google calender by clicking the sync button. Note: "School" task type will generate the typical 8 hour school blocks Monday - Friday.

Made on Flask using Python and Java.
